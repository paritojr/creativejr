# contributing
you can contribute in different ways, here are some of them:

## pull requests 
if it's a bug fix, in that case show the bug and a screenshot of the fix<br>
if it's a new feature, in that case demonstrate the feature through a screenshot

## issues
if you found a bug, open an issue containing:
- what the bug does
- how to trigger the bug
- a screenshot of it
- in what tool(s) does it appear

once you've provided these details, we can now inspect and see what's wrong with the tool
